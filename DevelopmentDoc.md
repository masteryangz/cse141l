### 2024.4.24  

ISA size:
 - 9 bits  


Register:
- size: 1 byte
- total amount: 8 
- namespace: 3 bits

Instructions:
- total amount: 8 
- namespace: 3 bits
- add
- and
- shf: by default left shift
- nop: no operation
- xor: bitwise xor
- ld: load an intermediate to a reg
- beq: take branch when operand==0
- pos: taken branch when operand>0
- lw: load from memory at reg
- st: store to memory at reg
